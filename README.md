# DÉMARRAGE DU PROJET

Ce projet a été créé à l’aide de **Create React App**, un outil permettant de configurer rapidement une application React avec une configuration prête à l’emploi.

## PRÉREQUIS

- Node.js  
- npm ou yarn  

## COMMANDES DISPONIBLES

Dans le répertoire du projet, vous pouvez exécuter les commandes suivantes :

### npm start

Démarre l’application en mode développement.  
L’application est accessible à l’adresse :  
http://localhost:3000

La page se recharge automatiquement lors des modifications du code.  
Les erreurs éventuelles sont affichées dans la console.

### npm test

Lance les tests en mode interactif.  
Consultez la documentation officielle pour plus d’informations sur l’exécution des tests.

### npm run build

Génère une version optimisée de l’application pour la production dans le dossier `build`.

- Code minifié  
- Fichiers avec hachage  
- Application prête au déploiement  

### npm run eject

⚠️ **ATTENTION : CETTE COMMANDE EST IRRÉVERSIBLE.**

Elle permet d’accéder à l’ensemble des fichiers de configuration (Webpack, Babel, ESLint, etc.) afin de les personnaliser.  
Cette étape n’est généralement pas nécessaire pour les projets standards.
